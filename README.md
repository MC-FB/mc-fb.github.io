1. **Clone the repository:**
   ```bash

   cd personal-website
   ```

2. **Open in your browser:**
   - Simply open `index.html` in your web browser
   - Or use a local server for better development experience

3. **Customize content:**
   - Replace placeholder text with your information
   - Update images in the HTML files
   - Modify colors in `styles.css` if needed

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #8B5A2B;     /* Main brown color */
    --secondary-color: #CD853F;   /* Secondary brown */
    --accent-color: #D2691E;      /* Accent color */
    --text-color: #3A2F2F;        /* Dark text */
    --light-text: #F5F5DC;        /* Light text */
    --background: #FFF8DC;        /* Background */
}
```
